<HTML> Hypertext Markup Language - Teil 1
Erstelle einen Text, welcher dein Lieblingstier beschreibt. Versuche, dabei das Tier so gut wie möglich zu beschreiben, dass du dessen Namen nicht erwähnen musst. Der Text sollte aus Überschrift(en) und Paragraphen bestehen. Hebe wichtige Aspekte fett/kursiv hervor.
Platziere am Schluss ein Bild des Tieres, welches als Link auf dessen Wikipediaseite verweist.

CSS Cascading Style Sheets - Teil 2
Style deine Tabelle mit Farben/Border/Padding etc. nach deinen Wünschen. Erstelle eine Werbung, welche immer am gleichen Ort angezeigt wird, egal wie weit man auf der Seite herunterscrollt. Gebe deinem Bild eine relative/absolute Positionierung und verschiebe es auf der Seite herum. Beschreibe in einem HTML-Kommentar der Unterschied der beiden Attribute. Unterteile deine gesamte bisherige Seite in DIVs (Divisions). Dabei sollten jeweils alle Teile, die inhaltlich zusammengehören im selben DIV platziert werden. DIVs können auch verschachtelt werden. Gebe allen DIVs, die gleich gestylt werden sollen jeweils dieselbe Klasse. Gebe DIVs, die spezifisch gestylt werden sollen eine ID. Gebe allen Klassen und IDs verschiedene Attribute.

Javascript - Teil 1
Erstelle zuerst drei Dokumente: HTML, CSS, JS und verknüpfe diese untereinander. Erstelle dann in deinem HTML-Dokument ein Formular mit einem Inputtextfeld und einem Submit-Knopf. Auf submit soll der Text aus dem Inputfeld als alert angezeigt werden. Das gesamte Formular kann der in "onsubmit" definierten Funktion als Parameter "this" mitgegeben werden (bspw.: onsubmit="myFunction(this)"). Die einzelnen Inputfelder können dann über deren jeweiliges name-Attribut abgerufen werden: this.name.value.

Javascript - Teil 3
Erstelle ein Objekt, welches dieselben Eigenschaften wie die Tabelle auf der Parlamentarierseite hat. Das Objekt sollte eine Funktion haben, die zurückgibt ob es sich beim Parlamentarier um einen Ständerat handelt. Style alle Ständeräte in einer Farbe und alle Nationalräte in einer anderen Farbe (Siehe unten, wie du ein deiner Homepage mit JavaScript ein neues Element hinzufügen und stylen kannst).
Wähle vier deiner Lieblingskantone und schreibe eine Funktion, die jeweils die Parlamentarier anders stylt, je nachdem in welchem Kanton sie wohnen, und ihnen einen Default-Style gibt, falls sie nicht in einem deiner Lieblingskantone wohnen. Erstelle ein Formular, das jeweils auf Submit ein Parlamentarier-Objekt erstellt, die oben genannten Funktionen überprüft und das Objekt ausgibt.

Javascript - Teil 4
Zu jedem erstellten Objekt soll jeweils das Erstelldatum hinzugefügt und das ganze Objekt in einem Array gespeichert werden. Das Array darf höchstens eine gewisse (von dir gewählte) Länge haben und der älteste Beitrag soll jeweils beim Überschreiten der Länge gelöscht werden, bevor der neuste Beitrag hinzugefügt wird. Gebe zuerst das ganze Array aus und dann einen zufälligen Eintrag davon.

JQuery - Teil 1
Lies unten nach, wie mit jQuery neue Objekte dem DOM hinzugefügt werden können. Füge alle Elemente des Array dem DOM hinzu. Dafür musst du dem Objekt eine toString()-Methode hinzufügen, die das Objekt schön darstellt. Beim Erstellen eines neuen Objekts sollte das Objekt mit einem fadeIn Effekt dem DOM hinzugefügt werden. Das älteste Objekt, das aus dem Array gelöscht wird, sollte auch mit einem Effekt aus dem DOM entfernt werden. (Du kannst den neu erstellen Elementen mit attr() eine Klasse oder einmalige ID geben, um sie wiederzufinden)

JQuery - Teil 2
Erstelle eine Funktion, die deine Objekte in einen String umwandelt, der dann per AJAX-Request einen Eintrag auf der Parlamentarierseite abrufen kann. Die AJAX-Methode sollte dir einen JSON-String zurückgeben, stelle diesen dann ansprechend auf deinem DOM dar. Am Ende solltest du ein Suchformular haben, das auf Submit die Daten der Parlamentarierseite abruft, und dir alle auf die Suche zutreffenden Einträge zurückliefert und darstellt.
